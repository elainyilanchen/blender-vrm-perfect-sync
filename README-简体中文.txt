=====================================================
 VRM 完美同步小工具 - 快速说明(纯文本版)
=====================================================

这是什么
  给 VRoid 的 VRM 模型批量添加 iPhone/ARKit「完美同步」
  (52 个表情形态键),无需任何 Blender 基础。
  不会修改原文件 - 结果另存为  模型名_ps.vrm

如何开始
  1. 把本文件夹解压到一个 [短路径],例如:  C:rm-tool
     (不要放在很深的文件夹里,比如微信接收文件夹 -
      Windows 路径超过 260 字符会出错)
  2. 双击  Perfect-Sync.bat
     - 完整版(Full):直接启动,所有依赖已打包。
     - 轻量版(Lite):缺什么(Python / Blender / VRM 插件),
       首次运行的向导窗口可以一键下载。
       错过了那个窗口?点应用右上角的「初始化环境...」。
  3. 在应用里:
     - 左边 = 供体:一个已做好完美同步的模型
              (samples 文件夹里有现成的)
     - 右边 = 受体:要处理的模型(可多选)
     - 选好输出文件夹,点「开始处理」。
  4. 把新生成的 *_ps.vrm 导入面捕软件即可。完成!

重要 - VRoid Studio 导出设置
  - 导出选 VRM 0.0(VRM 1.0 会被拒收)
  - 不要勾选「减少多边形」和「删除透明网格」
    (会改变脸部结构,导致表情无法转移)
